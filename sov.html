SOV website

Web

專案說明

為舒服美公司官方網站，接手後主要為功能性新增

部落格頁面

Propose:

部落格頁面除了做衛教傳達外，也是網頁提高SEO分數的主力文章區域，利用在搜尋引擎上的曝光來提高進站流量，增加消費者下單的機率

Progress:

  1. 定義功能目的

    在設計前的提案會議中訂定了一些方向

    ➊ 從後台上稿

    ➋ 流量主要來自於外部連結

    ➌ 有固定的 Grid

  2. Wireframing

    項目開始時就遇到一些問題

    ➊ 文章分類

    雖然文章大方向都與矯正有關，不過跟矯正有關的議題有很多，理想上會想利用文章分類的方式讓使用者可以快速找到自己需要的知識，但在尚未有任何文章排程計畫的情況下，不太適用頁籤的分類模式，因此這邊採取比較彈性化的作法，利用 Hashtag 標籤的方式進行

✅

UX 官網重構

Propose:

欲藉由提高消費者在站內瀏覽的時間來增加留單機率

Progress:

與外部 UX 工作室合作，由對方進行前期問卷調查，待前期準備工作完成後，再由我這邊負責進行頁面的 UI 設計。

問卷調查

透過問卷調查，將消費者的狀態區分為

  1. 下定決心要矯正
  2. 選擇矯正的自我評估
  3. 開始矯正的過程
  4. 矯正結束

考慮到最終目的是留單量，因此將優化重點放在前面提高消費者想矯正的契機上，
